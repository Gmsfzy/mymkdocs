# 欢迎来到我的文档站！

完整文档请访问 [mkdocs.org](https://www.mkdocs.org).

这是一个由 MkDocs + Material 主题构建的静态网站。

## 指令

* `mkdocs new [dir-name]` - 创建一个新项目。
* `mkdocs serve` - 启动实时加载文档服务器。
* `mkdocs build` - 建立文档网站。
* `mkdocs -h` - 打印帮助信息并退出。
